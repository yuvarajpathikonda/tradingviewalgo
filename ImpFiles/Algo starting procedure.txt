Delet the old access token and Generate new access token
Copy the access token, Go to .env file in notepad++ and replace the token(DHAN_ACCESS_TOKEN=)
Change the expiry code as required (EXPIRY_CODE) 0 for current/near expiry, 1 for next expiry and 2 for far expiry
Open cmd prompt and copy paste below commands
